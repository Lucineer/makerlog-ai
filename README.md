# Build Something Amazing

> MakerLog.ai — AI assistant for makers, builders, and DIY projects. Project planning, parts lists, tutorials. Part of the Lucineer ecosystem.

## What It Does

- Project planning and milestone tracking
- Parts lists with sourcing suggestions
- Step-by-step tutorial generation
- Troubleshooting from photos/descriptions
- Bill of materials cost estimation
- Community project sharing

## How It Works

makerlog-ai is a themed Cocapn instance. Behind the scenes:

- **Equipment** — What it perceives: your notes, documents, inputs
- **Skills** — How it thinks: domain expertise, reasoning patterns
- **Context** — What it remembers: conversation history, progress

## The Ecosystem

- **Layer 1 — Touch:** makerlog-ai, studylog.ai, activelog.ai, dmlog.ai — "It just works."
- **Layer 2 — Operate:** cocapn.ai / cocapn.com — "Customize and manage."
- **Layer 3 — Build:** deckboss.ai / deckboss.net / capitaine.ai — "Design systems, open the hood."
