Get Started in 60 Seconds

1

Fork and clone

git clone https://github.com/Lucineer/makerlog-ai.git && cd makerlog-ai
2

Set your API key

export ANTHROPIC_API_KEY=sk-ant-... # or OPENAI_API_KEY, DEEPSEEK_API_KEY, etc.
3

Run it

npm install && npm run dev # → http://localhost:8787
Open the IDE

Or continue in your terminal. The IDE is optional — everything works from CLI.
